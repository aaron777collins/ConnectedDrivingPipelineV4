# Pipeline Configuration Files

This directory contains JSON configuration files for the DaskPipelineRunner.

## File Naming Convention

Configs are named based on pipeline parameters:
```
pipeline_[distance]_[attack_type]_[features]_[location].json
```

Examples:
- `pipeline_2000m_rand_offset_100_200_ext_timestamps_n106_41.json`
- `pipeline_1000m_position_swap_xy_elev_n106_41.json`
- `pipeline_500m_const_offset_per_id_50_100_all_features_n105_41.json`

## Config Schema

All configs must conform to the DaskPipelineRunner schema. See the main README.md for the complete JSON structure.
